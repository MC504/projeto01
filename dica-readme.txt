O programa dica.c faz a validação de uma matriz 9x9 sugerindo possíveis soluções para elementos X de uma matriz. 
Essa matriz é lida através do teclado ou dando um "copiar" - "colar" de entradas válidas.
As células da matriz que são incognitas devem ser substituidas pelo caracter "X" (MAIÚSCULO)

Exemplo de entrada válida:
X 6 1 X X X X 9 X
4 3 X X 9 5 X 6 X
9 X X 1 X 2 X X 3
X X X 4 X 1 X X 9
5 X 9 X X X 7 X 1
6 X X 2 X 9 X X X
3 X X 9 X 8 X X 6
X 8 X 7 3 X X 4 2
X 9 X X X X 3 1 X

Após a entrada digitada basta apertar "Enter" para realizar a verificação das dicas. A verificação executa apenas uma vez.
